# Next-Chat
This application is a fun application showing the full use of Next.JS with an express server with full authentication using a mongoDB database. 

## Technologies

  1. Next.JS
  2. MongoDB
  3. Passport
  4. Express
  5. Node
  6. MaterilUI

## Default .env
  ```
    MONGO_URI=
    SESSION_SECRET=
    PRODUCTION_URL=
    PORT=3000
    NODE_ENV=

  ```
